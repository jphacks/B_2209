# ==== Emojis and Prefix ====
# :sparkles: feat:      A new feature
# :bug: fix:            A bug fix
# :books: docs:         Documentation only changes
# :lipstick: style:     Changes that do not affect the meaning of the code (white-space, formatting, missing semi-colons, etc)
# :recycle: refactor:   A code change that neither fixes a bug nor adds a feature
# :zap: perf:           A code change that improves performance
# :test_tube: test:     Adding missing or correcting existing tests
# :wrench: chore:       Changes to the build process or auxiliary tools and libraries such as documentation generation
# :wastebasket: remove: Remove files or codes
# :construction: WIP:   Work in Progress

# ==== Format ====
# (:emoji:)[prefix]: Subject:
#
# Commit body...
# (:emoji: は任意)

# ==== Example ====
# :sparkles: feat: TOPに戻るボタンを追加
